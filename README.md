# Zeely AI Sidebar

React + Vite + TypeScript UI component for background generation drawer.

## Requirements

- Node.js 18+

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build locally |
| `npm run lint` | Run ESLint |
| `npm run format` | Format with Prettier |

## Deploy

Build output is in `dist/`. Deploy to Vercel by importing the repo — build command and output directory are detected automatically.
